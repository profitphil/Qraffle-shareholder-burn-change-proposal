# Qraffle-shareholder-burn-change-proposal
Qraffle shareholder/burn % change proposal

## Proposal Overview

This is a formal governance proposal for the **Qraffle** smart contract system on the Qubic network.

### Current Revenue Distribution (as per current implementation)

From each raffle entry / proceeds:
- **Burn**: 10% — permanently removed from circulation (deflationary mechanism for $QUBIC)
- **Shareholders**: 3% — distributed as dividends to Qraffle smart contract shareholders
- **DAO / Registered members**: 5% (distributed to registered participants / DAO voters)
- **Charity (Qubic Charity Fund)**: 1%
- **Dev Fees**: 1%
- **Winner(s)**: remaining portion 80%

This structure has helped drive consistent burns and ecosystem support, but shareholder rewards have remained relatively low compared to the deflationary impact.

### Proposed Changes

Update the revenue allocation percentages as follows:

- **Shareholders**: **8%** (up from 3%) — significantly increased passive income for long-term holders and incentivizes more share ownership / investment in the Qraffle contract
- **Burn**: **5%** (down from 10%) — still provides meaningful deflation, but redirects more value to active participants and long-term supporters

All other allocations (DAO/registered members 5%, Charity 1%, Winner pool, etc.) remain unchanged unless a separate proposal addresses them.

### Summary of Change

| Category          | Current % | Proposed % | Difference | Rationale |
|-------------------|-----------|------------|------------|-----------|
| Shareholders      | 3%        | **8%**     | +5%        | Better rewards long-term holders, increases incentive to buy/hold shares, improves contract attractiveness |
| Burn              | 10%       | **5%**     | -5%        | Maintains deflationary pressure while reallocating value to ecosystem participants |

### Voting Options

- **Option 0 - NO** — Keep current distribution (**Shareholders 3%**, **Burn 10%**)
- **Option 1 - YES** — Approve the change to **Shareholders 8%** and **Burn 5%**
